# SIEM Data Analysis Thesis Repository
A work in progress repository for the analysis of the Advanced SIEM Dataset, a synthetic collection of security event records designed for training machine learning and artificial intelligence models in cybersecurity. The repository contains materials and documentation related to the dataset, including explanations of key terms and concepts, as well as potential analytical approaches that could be applied to the dataset for a master's thesis.


SIEM Dataset: [text](https://huggingface.co/datasets/darkknight25/Advanced_SIEM_Dataset)

---

## Repository structure: 
The repository is organized into the following files:

- `readme.md`: This file, providing an overview of the repository and its contents.

- `termsExplanations.md`: A detailed explanation of the key terms and concepts that will be used in the thesis, such as different machine learning algorithms, statistical methods, and domain-specific frameworks relevant to SIEM data analysis.
- `whatWeCouldDo.md`: A comprehensive outline of the potential analyses and methodologies that could be applied to the SIEM dataset, including data engineering techniques, supervised and unsupervised learning approaches, NLP applications, and advanced analytics like UEBA and MITRE mapping.

- *FOLDER* `proposals/`: This folder contains many separate files, each detailing a distinct thesis proposal based on the SIEM dataset. Each proposal includes an abstract, problem statement, proposed architecture, research objectives, implementation stack, and expected contributions.
- `proposals/GuideToTheProposals.md`: Brief summaries of each proposal, highlighting their unique focus areas and methodologies, to help guide the selection process for the thesis topic.7
